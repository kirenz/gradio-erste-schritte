# AGENTS.md

## Projektübersicht

Dies ist ein Lernprojekt für **Gradio-Basics**. 

## Technologie-Stack

- **Gradio 6.1.0** - Python-Framework für schnelle UI-Entwicklung von ML/KI-Anwendungen
- **Python 3.11+** - Programmiersprache

## Paketverwaltung

Dieses Projekt verwendet **uv** als Paketmanager.

### Pakete installieren

```bash
uv add <paketname>
```

### Projekt einrichten

```bash
uv sync
```

### Projekt ausführen

```bash
uv run python <dateiname>.py
```

## Code-Stil

- Verwende deutsche Kommentare für Lernzwecke
- Halte den Code einfach und gut dokumentiert
- Jede Datei sollte als eigenständiges Beispiel funktionieren
